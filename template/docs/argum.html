Here lists all arguments of rwcap.

Name Default Description
-f None The input file name, required.
-p None The threshold of relative error of Self capacitance.
-c "" The threshold of relative error of coupling capacitance. *
-t 1000 The number of walks for termination.
-n 1 The number of threads for parallel computing.
-w None The minimum wire width of the process technology.
-o "rwcap.out" The output file name.
--std-output "enable" Enable or disbale output to stdout.
--file-output "enable" Enable or disbale output to file.
--log-level "warn" The lowest level of log, should be one of debug|info|warn|error|disable.
--log-files "" The files for log, should be like "DEBUG,INFO=file1, WARN=file2, ERROR=stderr".
--ignore-medium None Ignore <medium> field of the input.
--rewrite-tasks None Override the <task> field of the input.
--single-table None The path to single dielectric GFTs, override the registered path.
--twoly-table None The path to medium-dielectric GFTs, override the registered path.
--spacemgr-type "hybrid" The type of space management structure, should be one of hybrid|octree|ongrid. *
--spacemgr-candlen 20 for hybrid
20 for octree
40 for ongird		 The max number of blocks in a spacial cell.
--spacemgr-vector "disable" Speedup the lookup in space management with vectorization.
--gauss-scale 1.2 The gauss-scale factor.
--gauss-type "diel" The sampling PDF on Gaussian surface, should be one of uniform|diel|dist|both.
--rotated-cube "enable" Rotate the transition domain cube when it's near a non-Manhattan block.
--as-manhattan 0 Use square prism to approximate cylinder, should be 0 or 1.
* Each coupling threshold should like "netA netB limit error", which requires: ErrAB≤error*max(1, limit*CapA/CapAB).
* The refered papers may help to understand arguments about space management, Gaussian surface and non-Manhattan block.
