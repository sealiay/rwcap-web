Here lists all arguments of rwcap.

Name Default Description
-f None The input file name, required. *
-p None The threshold of relative error of self capacitance.
-t 1000 The number of walks for termination. This is ignored if '-p' is set.
-n 1 The number of threads for parallel computing.
-w None The minimum wire width of the process technology.
-o "rwcap.out" The output file name.
--std-output "enable" Enable or disbale output to stdout.
--extract-level "net" Report the capatance between nets or blocks, should be one of "net"|"block".
--coupling-error "" The threshold of relative error of coupling capacitance. *
--log-level "warn" The lowest level of log, should be one of "debug"|"info"|"warn"|"error"|"disable".
--log-debug-file "stdout" The file for debug level log.
--log-info-file "stdout" The file for info level log.
--log-warn-file "stderr" The file for warn level log.
--log-error-file "stderr" The file for error level log.
--multi-dielectric "enable" When disabled, the <medium> field of the input will be ignored.
--overwrite-tasks None Overwrite the <task> field of the input.
--table-single None The path to single dielectric GFTs, overwrites the registered path.
--table-twolyr None The path to medium-dielectric GFTs, overwrites the registered path.
--spacemgr-type "hybrid" The type of space management structure, should be one of "hybrid"|"octree"|"grid". *
--spacemgr-candlen None The max number of blocks in a spacial cell.
--gauss-scale 1.2 The gauss-scale factor.
--gauss-type "diel" The sampling PDF on Gaussian surface, should be one of "uniform"|"diel"|"dist"|"both".
--strat-sampling "enable" Enable or disable stratified sampling.
--rotated-cube "enable" Rotate the transition domain cube when it's near a non-Manhattan block.
--nonmanhattan-spacemgr "enable" Enable or distance the space manager for nonManhattan blocks
--manhattan-approximation "disable" When enabled, square prism is used to approximate cylinder.
* The file name in arguments can also be "stdin", "stdout", "stderr", "null" (similar to "/dev/null"), "-" (automatically choose stdin or stdout).
* Each coupling threshold should like "netA netB limit error", which requires: ErrAB≤error*max(1, limit*CapA/CapAB).
* The refered papers may help to understand arguments about space management, Gaussian surface and non-Manhattan block.
