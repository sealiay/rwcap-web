This program generates the multi-dielectric surface Green's function table for multi-layered VLSI process technology.

Once downloaded, unzip the package. You'll see something like this:

techGFT/
├── GFT_Gen.p
├── README
├── RunTechGFT.p
├── example.diel_conf
└── Generator2Diel/

The input file should be named as name.diel_conf. Take example.diel_conf as an example.
Line starts with ".d" indicates a dielectric interface, followed by the relative permittivity of its two sides.
Line starts with "*" is comment.

In Matlab command line, run RunTechGFT.p then input the file name, or directly run GFT_Gen.p <file_name> to execute it.
