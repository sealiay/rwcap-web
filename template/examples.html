Examples

Get started to know RWCap from here.

Case 1.

This case only contains a unit cube. We will extract its capacitance.

Download the RWCap-solver and the case files (here) and unzip them.
Open cases/cube.cap3d, it looks like this:

Run it with rwcap-solver/rwcap -f cases/cube.cap3d -p 0.01 to get the capacitance with 1-σ error ≤ 1%.

Case 2.

This case contains two layers of parallel wires.

Open cases/wires.cap3d, it looks like this:

This is a multi-dielectric case which requires compatible Green's function tables. You need to merge cases/tables into rwcap-solver/tables.
Then run it with rwcap-solver/rwcap -f cases/wires.cap3d -p 0.005 -o wires.out to get the capacitance of all nets with 1-σ error ≤ 0.5%.

The result in wires.out looks like this:

For each master net, the line starts with 'Master <name>' shows its self-capacitance and error. Following lines are the coupling capacitance related to this master net.

When running, there will be a status line shows current master net's self capacitance and the progress percentage after every 100,000 walks. It looks like this:

Case 3.

This case shows how to generate the Green's function tables.

Download the TechGFT tool (here) and unzip it.
Open cases/sample.diel_conf, it looks like this:

Open Matlab and excute RunTechGFT.p then input the file name sample.diel_conf.
The results would be under sample_GFT (same as the input file name) directory.

The new generated GFTs should be registered with rwcap-solver/rwcap --gft <some where>/sample_GFT/.
If you run rwcap-solver/rwcap -f cases/wires.cap3d -p 0.01 again, it now use the new tables.
